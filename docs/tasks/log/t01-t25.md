### T07 — 弯头几何计算器 (2026-03-28)

**产出文件**: `src/engine/BendCalculator.h/.cpp` · `tests/test_engine.cpp`

**接口**: → `src/engine/BendCalculator.h`

**设计决策**:
- θ = angle(d1, d2)；tangentLength = R × tan(θ/2)；N = A06 - d1×tanLen, F = A06 + d2×tanLen
- 弧圆心：C = N + cross(normal, d1) × R；弧中点：M = C + normalize(A06-C) × R
- θ < 1e-6 rad（直线）或 θ > π-1e-6 rad（U-turn）或输入点重合时返回 nullopt
- 使用 OCCT gp_Pnt/gp_Vec 直接计算

**已知限制**:
- 不支持 θ=0（直线）和 θ=π（U-turn）退化情况
- 未考虑管壁厚度对弯弧偏移的影响

### T08 — 管件几何生成器 Run/Reducer/Tee (2026-03-28)

**产出文件**: `src/engine/RunBuilder.h/.cpp` · `BendBuilder.h/.cpp` · `ReducerBuilder.h/.cpp` · `TeeBuilder.h/.cpp` · `GeometryDeriver.h/.cpp` · `tests/test_pipe_geometry.cpp`

**接口**: → `src/engine/RunBuilder.h`, `BendBuilder.h`, `ReducerBuilder.h`, `TeeBuilder.h`, `GeometryDeriver.h`

**设计决策**:
- RunBuilder: `BRepPrimAPI_MakeCylinder(ax, r, length)` 直接在方向轴上构建
- BendBuilder: `BRepPrimAPI_MakeTorus(ax, majorR, minorR, angle)` 截段
- ReducerBuilder: `BRepPrimAPI_MakeCone(ax, r1, r2, length)`；内外锥 cut 得到锥壳
- TeeBuilder: 先 fuse 实心外圆柱，再 fuse 实心内圆柱，最后 cut — 避免直接 fuse 薄壳超时
- GeometryDeriver: Reducer 的 endOD 从 `typeParams["endOD"]`；Tee 支管端点从 `typeParams["branchEndX/Y/Z"]`

**已知限制**:
- TeeBuilder boolean fuse 在 Debug 构建中每次约 1s（Release 会快很多）
- TeeBuilder 中 branchPoint 应在主管中心线上，否则几何可能不正确

### T09 — 管件几何 (Valve/FlexJoint/Beam/Accessory) (2026-03-28)

**产出文件**: `src/engine/ValveBuilder.h/.cpp` · `FlexJointBuilder.h/.cpp` · `BeamBuilder.h/.cpp` · `AccessoryBuilder.h/.cpp` · `tests/test_valve_flex_beam.cpp`

**接口**: → `src/engine/ValveBuilder.h`, `FlexJointBuilder.h`, `BeamBuilder.h`, `AccessoryBuilder.h`

**设计决策**:
- ValveBuilder: 3 段实心圆柱 fuse 后 cut 内腔；gate 阀体直径 1.8×OD，ball/check 为 1.6×OD
- FlexJointBuilder: 交替扩缩锥体 fuse 生成波纹外形，再 cut 内径圆柱；膨胀比 1.5×OD
- BeamBuilder: BRepBuilderAPI_MakeFace + BRepPrimAPI_MakePrism 挤出截面
- GeometryDeriver: Valve 从 `typeParams["valveType"]`，FlexJoint 从 `typeParams["segmentCount"]`

**已知限制**:
- FlexJointBuilder 波纹数 >5 时 Debug 下 fuse 耗时增加（每段约 80-100ms）
- BeamBuilder 仅支持直梁
