# M5 异步重算落地

## T71 重构 RecomputeEngine 异步管线

**目标**
将重算逻辑改为“主线程收集、后台计算、主线程提交”的管线。

**依赖**
- T62
- T67
- T68
- T69
- T70

**交付物**
- 异步重算主链路
- 重算结果提交机制

**验收标准**
- `RecomputeEngine` 不直接承担线程管理细节

## T72 后台化 ShapeMesher 与批量重算

**目标**
优先处理最明确的性能热点。

**依赖**
- T71

**交付物**
- 后台网格化执行路径
- 批量重算后台执行路径

**验收标准**

**依赖**
- T68

**验收标准**
